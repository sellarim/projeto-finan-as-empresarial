# projeto-finan-as-empresarial
Análise estratégica em Python para diagnosticar a saúde financeira de um programa de fidelidade. Detectamos que a massiva migração para pagamentos de boletos drenou o caixa, gerando prejuízo operacional. O projeto utiliza Pandas e Seaborn para visualizar a correlação crítica entre a mudança de comportamento do usuário e a quebra do fluxo de caixa.
